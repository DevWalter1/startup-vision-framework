## Template 6: Der Test

### Test 1: Der Bar-Test

Erkläre deine Vision/Mission jemandem in einer Bar (oder einem Freund/Familienmitglied). Versteht er/sie es sofort?

**Feedback:**
```
Was hat die Person verstanden?



Was war unklar?



```

### Test 2: Der 5-Jahres-Test

Wird eure Vision in 5 Jahren noch relevant sein? Wird eure Mission in 5 Jahren noch zutreffen?

**Vision:**
```
Relevant in 5 Jahren? [ ] Ja [ ] Nein
Warum?



```

**Mission:**
```
Noch zutreffend in 5 Jahren? [ ] Ja [ ] Nein
Warum?



```

### Test 3: Der Team-Test

Würdest du stolz sein, diese Vision/Mission einem potenziellen Mitarbeiter zu zeigen? Würde er/sie sich davon inspirieren lassen?

**Ehrliche Antwort:**
```
[ ] Ja, definitiv
[ ] Ja, aber...
[ ] Eher nicht

Was fehlt?



```
